🎨 SECURITY LEVELS
═══════════════════════════════════════════════════════════════════════════

🟢 SECURE
  • 12-16 characters
  • Uppercase + Lowercase + Numbers + Symbols
  • No obvious patterns
  Example: MyStr0ng@Pwd99

🟠 HALF-SECURE  
  • 7-11 characters
  • Uppercase + Lowercase + Numbers + Symbols
  • No obvious patterns
  Example: MyP@ss99

🔴 UNSAFE
  • Less than 7 characters
  • Missing any complexity requirement
  • Contains obvious patterns
